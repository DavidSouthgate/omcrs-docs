# Usage
## Requirements for Development
 * [NPM](https://www.npmjs.com/)
 * [node.js](https://nodejs.org/en/)

## Running in Development
Once requirements are installed, open a terminal in the project hone.
(The same directory as main.js)

Run the following command to install dependencies:
```
npm install
```

Run the following command to run the program
```
npm start
```

## Compiling for Distribution
Run the following command to compile the application for all systems
that this system supports
```
npm run dist
```

This will generate multiple different binary files in the `dist`
directory.
